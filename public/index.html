Image Search Abstraction Layer

  • Pass /api/TEXT as a parameter.
    This will retrieve a JSON response of image URLs, alt text, and page URLs relating to TEXT.
    Optional: append ?offset=NUMBER to paginate NUMBER value responses.

  • Pass latest as a paramter.
    This will return a JSON response of the most recently submitted search images

Example usage

https://search-img.glitch.me/api/cat

https://search-img.glitch.me/api/cat?offset=10

https://search-img.glitch.me/latest
